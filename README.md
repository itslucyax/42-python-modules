## Module 00 — Growing Code

Introduction to Python fundamentals through community garden scenarios.

| Exercise | Function | Concepts |
|----------|----------|----------|
| ex00 | `ft_hello_garden` | def, print |
| ex01 | `ft_garden_name` | input, variables, f-strings |
| ex02 | `ft_plot_area` | int(), arithmetic |
| ex03 | `ft_harvest_total` | multiple inputs, sum |
| ex04 | `ft_plant_age` | if / else |
| ex05 | `ft_water_reminder` | if / else |
| ex06 | `ft_count_harvest` | for, range, recursion |
| ex07 | `ft_seed_inventory` | type hints, mypy, string methods |

## Setup

```bash
git clone https://github.com/itslucyax/42-python-modules.git
cd 42-python-modules/module_00
python3 main.py
```

## Norms

- Python 3.10+
- [flake8](https://flake8.pycqa.org/) compliant
- [mypy](https://mypy-lang.org/) for type checking (module 00, ex07+)
